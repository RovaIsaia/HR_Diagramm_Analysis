# Analyse and Generate an HR Diagram

A code package to analyze a dataset (6-class CSV) and generate a Hertzsprung-Russell diagram based on this dataset.

## Contents

1. `6_class_csv.csv` – The dataset.
2. `requirements.txt` – List of required dependencies.
3. `Data_Analysis_HR_Diagram.ipynb` – Jupyter notebook for analyzing the dataset.
4. `HR_Diagram_Maker.py` – Python script to generate the HR diagram locally.

## Steps to Use

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
2. Run the script to generate the HR diagram:
   ```bash
   python HR_Diagram_Maker.py
